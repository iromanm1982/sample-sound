# App Icon Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the default Android launcher icon with a custom flat equalizer-bars icon in purple.

**Architecture:** Two VectorDrawable XML files are replaced — the background (solid `#6200EE`) and the foreground (five vertical bars in `#BB86FC`/`#EAB8FF`). The existing adaptive icon manifest (`mipmap-anydpi/ic_launcher.xml`) already wires them up correctly and needs no changes. Since `minSdk = 30`, all devices render adaptive icons; the `.webp` raster fallbacks are unused.

**Tech Stack:** Android VectorDrawable XML, Adaptive Icon (API 26+)

---

### Task 1: Replace background drawable

**Files:**
- Modify: `app/src/main/res/drawable/ic_launcher_background.xml`

The current file contains a green grid (default Android template). Replace it entirely with a solid purple fill.

- [ ] **Step 1: Overwrite `ic_launcher_background.xml`**

Replace the full content of `app/src/main/res/drawable/ic_launcher_background.xml` with:

```xml
<?xml version="1.0" encoding="utf-8"?>
<vector xmlns:android="http://schemas.android.com/apk/res/android"
    android:width="108dp"
    android:height="108dp"
    android:viewportWidth="108"
    android:viewportHeight="108">
    <path
        android:fillColor="#6200EE"
        android:pathData="M0,0h108v108h-108z" />
</vector>
```

- [ ] **Step 2: Commit**

### Task 2: Replace foreground drawable with equalizer bars

**Files:**
- Modify: `app/src/main/res/drawable/ic_launcher_foreground.xml`

The current file contains the default Android robot. Replace it with five vertical equalizer bars.

**Layout math** (all values in dp within the 108×108 viewport):
- Bar width: 10dp — gap between bars: 6dp
- Total group width: 5×10 + 4×6 = 74dp → starts at x = 17dp
- Bottom padding: 18dp → bars aligned to y = 90dp
- Top-corner radius: 4dp
- Bar x positions: 17, 33, 49, 65, 81
- Bar top y values (y_bottom − height): 64, 48, 56, 40, 70

**Path formula for each bar** (rounded top corners only):

```
M x,(y_top+4)  L x,90  L (x+10),90  L (x+10),(y_top+4)
Q (x+10),y_top  (x+6),y_top
L (x+4),y_top
Q x,y_top  x,(y_top+4)  Z
```

- [ ] **Step 1: Overwrite `ic_launcher_foreground.xml`**
